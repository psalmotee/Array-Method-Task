Node.js Installation Guide

This guide will help you install Node.js and run a simple JavaScript file using Visual Studio Code.

How to Install Node.js

Node.js is a popular JavaScript runtime that allows you to run JavaScript code outside of a browser. To install Node.js on your computer, follow these steps:

  1. Go to the official Node.js website: https://nodejs.org
  2. Download the recommended version for your operating system.
  3. Run the installer and follow the setup instructions.
  4. After installation, open your command prompt or terminal and type node -v to check the installed version.

Running Node.js in VS Code

  1. Open Your Project Folder
    Launch VS Code.
    create a new folder.
  2. Create a JavaScript File
    Example: index.js
    console.log("Hello from Node.js!");
  3. Open Integrated Terminal
    Go to Terminal > New Terminal or press:
    Ctrl + Shift + `
  4. Run the File with Node
    In the terminal, type:
    node index.js
    You should see: 
    Hello from Node.js!

Once installed, you can start building applications using Node.js!
